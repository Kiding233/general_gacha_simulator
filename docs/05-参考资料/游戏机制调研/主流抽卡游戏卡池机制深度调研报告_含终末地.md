# 主流抽卡游戏卡池机制深度调研报告（含终末地）

**报告日期**：2026年5月26日  
**调研范围**：原神、崩坏：星穹铁道、绝区零、鸣潮、明日方舟、**明日方舟：终末地**、碧蓝档案、公主连结、阴阳师、火影忍者手游、FGO、赛马娘  
**关键词**：保底机制、概率曲线、资源反馈、天井系统、首付返利、星辉/星尘、黄票/绿票、珊瑚、寻访档案、武器配额

---

## 摘要

本报告对当前主流抽卡游戏的卡池机制进行了系统性深度调研，覆盖米哈游系（原神/星铁/绝区零）、库洛系（鸣潮）、鹰角网络（明日方舟/**终末地**）、悠星/Nexon（碧蓝档案）、Cygames（公主连结/赛马娘）、网易（阴阳师）、腾讯（火影忍者手游）及Aniplex（FGO）等厂商的核心产品。报告从**概率结构、保底机制、资源系统、继承规则、反馈循环**五个维度拆解每种机制的最细节实现，为抽卡决策建模提供真实世界参数基准。

## 1. 米哈游系：原神 / 崩坏：星穹铁道 / 绝区零

### 1.1 原神（Genshin Impact）

#### 1.1.1 概率曲线（精确参数）

| 参数 | 角色池 | 武器池 | 常驻池 |
|------|--------|--------|--------|
| **5星基础概率** | 0.6% | 0.7% | 0.6% |
| **软保底起始** | 第74抽 | 第63抽 | 第74抽 |
| **软保底增幅** | 每抽+6% | 每抽+6% | 每抽+6% |
| **硬保底** | 90抽 | 80抽 | 90抽 |
| **综合概率（含保底）** | 1.6% | 1.85% | 1.6% |
| **4星基础概率** | 5.1% | 6.0% | 5.1% |
| **4星硬保底** | 10抽 | 10抽 | 10抽 |

**软保底的精确概率递增**：citeweb_search:10#0web_search:10#16
- 第74抽：~32.4%（54倍基础率）
- 第75抽：~38.4%
- 第76抽：~44.4%
- 第80抽：~98.2%
- 第90抽：100%

**平均出货抽数**：约62-64抽（角色池），50-55抽（武器池）。仅~1-2%玩家触及硬保底。citeweb_search:10#1

#### 1.1.2 大小保底机制（50/50）

- **小保底**：首次5星有50%概率为UP角色，50%概率"歪"常驻5星
- **大保底**：若小保底歪了，下一次5星**必定**为UP角色
- **捕获明光（Capturing Radiance）**：5.0版本后，歪了时有10%概率静默升级为UP角色，实际UP率从50%提升至约55%citeweb_search:10#0
- **硬保底上限**：最非情况下180抽必得UP角色（90抽歪 + 90抽大保底）

#### 1.1.3 武器池定轨（Epitomized Path）

- 选择1把UP武器作为目标
- 首次5星武器有75%概率为UP武器之一（双UP池）
- 若未选中目标武器，获得1个**命定值（Fate Point）**
- **1个命定值即可保证下一次5星武器为选定目标**（5.0版本后简化）citeweb_search:10#0
- 命定值在卡池结束时清零，不继承

#### 1.1.4 资源产出与兑换系统（核心反馈机制）

**抽卡产物分级**：citeweb_search:11#0web_search:11#1

| 出货 rarity | 产物 | 无主的星辉 | 无主的星尘 |
|-------------|------|-----------|-----------|
| 3星武器 | 武器 | 0 | 15 |
| 4星角色（首次） | 角色 | 2 | 0 |
| 4星角色（重复） | 角色+命座 | 2 | 0 |
| 5星角色（首次） | 角色 | 10 | 0 |
| 5星角色（重复） | 角色+命座+星辉 | 10 | 0 |

**兑换比例（尘辉商店）**：citeweb_search:11#0web_search:11#25

| 兑换项 | 星辉价格 | 星尘价格 | 月度限制 |
|--------|----------|----------|----------|
| 纠缠之缘（限定池券） | 5星辉 | 75星尘 | 星尘限5个/月 |
| 相遇之缘（常驻池券） | 5星辉 | 75星尘 | 星尘限5个/月 |
| 单抽券（等值160原石） | — | — | — |

**关键反馈循环**：
- 抽限定池 $\rightarrow$ 产出星辉/星尘 $\rightarrow$ 兑换纠缠之缘 $\rightarrow$ 返回限定池
- **星辉兑换无限制**，星尘兑换每月限5个纠缠+5个相遇
- 3星武器大量产出星尘，构成"垫刀回本"的基础反馈

#### 1.1.5 继承规则

- **角色活动池**：保底计数完全继承，跨期保留；大保底状态继承
- **武器活动池**：保底计数继承，但**命定值不继承**（卡池结束清零）
- **常驻池**：独立计算，300抽可自选1个常驻5星角色
- **不同池子间不共享**计数器

| 参数 | 标准池 | 限定池 |
|------|--------|--------|
| **6星基础概率** | 2% | 2.5% |
| **软保底起始** | 50抽 | 100抽 |
| **软保底增幅** | 每抽+2% | 每抽+2% |

- 50抽后概率线性增长，第51抽为4%，第52抽为6%...第99抽为100%

### 3.2 保底与继承机制

- **常驻/标准池**：保底跨池共享（标准寻访之间互通）
- **限定池**：**独立计算**，不继承到其他池子（周年庆/半周年/春节限定各自独立）citeweb_search:10#15

| **源石** | 关卡首通/氪金 | 购买皮肤/兑换合成玉（1源石=180玉） | — |
| **资质凭证（绿票）** | 抽卡获得 | 兑换材料/单抽券 | — |
| **高级凭证（黄票）** | 抽卡/重复6星分解 | **258黄票 = 38抽**（最高性价比） | 约6.79票/抽 |

## 4. 鹰角网络：明日方舟：终末地（Arknights: Endfield）

### 4.1 角色池（干员寻访/Headhunting）

#### 4.1.1 概率曲线（精确参数）

| 参数 | 特许寻访（限定） | 标准寻访（常驻） | 新手寻访 |
|------|-----------------|-----------------|----------|
| **6星基础概率** | 0.8% | 0.8% | 0.8% |
| **5星基础概率** | 8% | 8% | 8% |
| **4星基础概率** | 91.2% | 91.2% | 91.2% |
| **10抽5星保底** | 有 | 有 | 有 |
| **软保底起始** | 65抽 | 50抽 | 无 |
| **软保底增幅** | 每抽+5% | 每抽递增 | 无 |
| **硬保底（小保底）** | 80抽必出6星 | 80抽必出6星 | 40抽必出6星 |
| **大保底** | 120抽必出UP | 无 | 无 |
| **自选保底** | 无 | 300抽自选标准6星 | 无 |

**关键差异**：终末地的6星基础概率（0.8%）高于明日方舟本体（2%），但硬保底从100抽缩短至80抽，且引入了独特的**双计数器并行系统**。citeweb_search:13#0web_search:13#9web_search:13#13

#### 4.1.2 双计数器并行保底系统（核心机制）

终末地采用**两个独立运作的保底计数器**，这是其与其他游戏最显著的区别：citeweb_search:13#0web_search:13#13web_search:13#16

**计数器1：80抽小保底（跨池继承）**
- 80抽内必出6星角色
- 该6星有50%概率为当期UP，50%概率为其他6星（标准池角色或往期限定）
- **此计数器跨池继承**：若在当前池子抽到79抽未出6星，下一池子第1抽必出6星
- 每次抽出6星后，该计数器重置为0
- **无"大保底继承"**：若80抽歪了，下一池子的80抽保底仍然是50/50，**不保证必出UP**

**计数器2：120抽大保底（不继承，池内单次）**
- 在**同一限定池内**，若120抽仍未获得当期UP角色，第120抽**必定**为该UP角色
- **此计数器不继承**：卡池结束后清零，无法带到下一池子
- 每个池子**仅触发一次**120保底
- 若在120抽前已抽出UP角色，该计数器失效

**计数器3：240抽信物保底（潜能/重复保底）**
- 获得首个UP角色后，累计240抽**必定**获得该UP角色的**信物**（用于解锁潜能/被动技能）
- 无论首个UP是在80抽还是120抽获得，240抽的信物保底始终从0开始计数
- 这是终末地**角色养成深度绑定抽卡**的核心设计citeweb_search:13#0web_search:13#13

#### 4.1.3 特殊福利机制

**30抽福利（Urgent Recruitment）**：
- 在限定池累计抽取30次后，赠送1次**紧急十连抽**
- **该十连不计入任何保底计数器**，仅作为额外奖励
- 享有基础概率（0.8%），但不占用80/120/240的进度citeweb_search:13#7web_search:13#13

**60抽福利（Headhunting Dossier/寻访档案）**：
- 在限定池累计抽取60次后，获得1份**寻访档案**
- 转化为**10张仅限下一期限定池使用的特殊抽卡券**
- **该券计入下期池子的保底计数器**
- 这是鹰角鼓励"跨期垫刀"的设计，将本期投入与下期收益绑定citeweb_search:13#0web_search:13#13

#### 4.1.4 卡池类型与UP规则

**特许寻访（Chartered Headhunting/限定池）**：
- 单UP池：当期1名6星UP角色
- **往期限定角色仍可在池子中抽出**，但概率与标准6星相同（非UP状态）
- 往期限定角色仅在**当前及下一期**限定池中出现，之后移出池子（绝版或等复刻）citeweb_search:13#14web_search:13#16

**标准寻访（Basic Headhunting/常驻池）**：
- 固定标准6星角色池
- 300抽可自选1名标准6星角色（终身一次）citeweb_search:13#6

**新手寻访（New Horizons Headhunting）**：
- 仅限新账号，终身一次
- 40抽必出标准6星角色
- 第40抽额外赠送**6星武器自选箱**
- 一旦抽出6星角色，该池子永久关闭citeweb_search:13#12

### 4.2 武器池（武库申领/Arsenal Exchange）

#### 4.2.1 概率与保底结构

| 参数 | 数值 |
|------|------|
| **6星武器基础概率** | 4% |
| **5星武器概率** | 15% |

## 11. 终末地机制的独特性与建模启示

### 11.1 双计数器并行系统

终末地的80抽（跨池继承）+ 120抽（池内不继承）设计创造了**独特的风险结构**：

- **安全垫刀区（0-80抽）**：玩家可以放心垫刀，因为计数器继承，沉没成本可控
- **高风险 commitment 区（80-120抽）**：一旦进入此区间，必须在池子结束前完成120抽，否则大保底清零
- **策略含义**：终末地强制玩家在"见好就收"（80抽前停）和"all-in到底"（至少120抽）之间做二元选择，**不存在"先垫40抽观望"的中间态**

### 11.2 角色-武器强制联动

武器配额系统使得**角色池投入直接决定武器池可用资源**：

- 数学上，角色池每抽的期望武器配额产出为：$0.008 \times 2000 + 0.08 \times 200 + 0.912 \times 20 + (1-0.008)^{10} \times 380 \approx 16 + 16 + 18.24 + 350 \approx 400$（十连期望）
- 武器池十连成本1980配额，即**每5次角色池十连可支持1次武器池十连**
- 这种设计强制玩家**必须同时规划两个池子**，无法独立决策

### 11.3 跨期投资激励（60抽档案）

60抽送下期10连券的设计，使得：
- 本期投入60抽的边际收益 = 当期出货期望 + 下期10连的期权价值
- 这创造了**非线性激励**：第59抽至第60抽的边际价值突然跃升（因为触发档案奖励）
- 玩家社区可能形成"垫到60抽"的共识策略，即使数学期望上50-60抽区间效率不高

### 11.4 往期限定的"缓冲期"设计

UP角色在当期及下一期池子均可抽出（非UP状态），这：
- 降低了FOMO（错失恐惧），因为玩家知道有"二次机会"
- 但增加了概率模型的复杂度：池子中存在**多个不同概率的6星目标**
- 对建模而言，池子的概率分布不再是"UP vs 非UP"二元，而是"当期UP（50%）vs 往期UP（低概率）vs 标准6星（低概率）"三元
