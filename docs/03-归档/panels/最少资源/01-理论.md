# 最少资源面板 -- 理论

> 创建日期：2026-06-02（首次建档）
> 关联模块：`gui/resource_search_panel.py`、`core/forward_backward.py`

---

## 一、核心假设

### 1.1 成功率对资源的单调性

二分搜索正确性的前提是**成功率随资源增加而单调不减**：

```
R₁ ≤ R₂ ⇒ P(success | resources=R₁) ≤ P(success | resources=R₂)
```

此假设在直觉上合理——更多资源意味着可以抽更多卡，从而增加达成目标的可能性。但存在潜在的违反场景：

- **保底计数器效应**：附加资源可能诱导策略消耗更多抽卡，意外触发保底计数器 → 反而降低成功率。这种情况在 smart 策略下不太可能发生（因为 smart 策略会选择最优行动），但在 fixed_count 或 pool_quota 等被动策略下可能发生
- **策略的离散决策边界**：某些策略在资源达到特定阈值时会改变行为（如「保底预留」策略在资源足够保底时转为激进抽卡），导致成功率在边界处非连续

**单调性在模拟层面**：即使单次模拟不满足单调性（由于随机种子波动），`compute_success_probability()` 的批量估计（N 次模拟的平均）应在 N 足够大时满足单调性（大数定律）。

### 1.2 翻倍法上界搜索的有限性

阶段一从 `initial_resource_hint`（或 `cost_per_draw × 100`）开始，每次翻倍直到成功率 >= 阈值。假设：

1. **存在有限上界**：任意目标卡集合都存在一个资源量使成功率 >= 阈值。对于「all_targets」GDR 且目标卡需求为 1 张时，可能不存在有限上界（如果某些卡池无限且 SSR 概率非 1.0，理论上有非零概率永远抽不到——但此概率指数递减）
2. **翻倍次数有限**：代码中最多翻倍 15 次。以初始值 100 抽（16000 资源）计算，15 次翻倍后为 100 × 2^15 = 3,276,800 抽——远超实际合理范围

### 1.3 二分搜索的最优性

经典二分法在最坏情况下需要 `⌈log₂((r_hi - r_lo) / epsilon)⌉` 次迭代才能收敛到 epsilon 精度。在目标函数为单调布尔判定（成功率 < 阈值 / >= 阈值）且无噪声时，二分法是最优的确定性搜索策略。

然而，成功率估计是**有噪声的**（随机模拟的采样误差），二分搜索的「判定」实际上是 `prob >= threshold` 的比较——这个比较有 Type I（将 < 阈值的资源误判为 >=）和 Type II（将 >= 阈值的资源误判为 <）错误。噪声可能使二分搜索在真实阈值附近「震荡」——当前代码依赖 `max_iterations` 来防止无限循环。

### 1.4 噪声对精度的影响

- **乐观偏差**：在某次迭代中，采样估计 `p̂` 可能高估 `p_true`，导致二分搜索过早接受一个较小的资源值（hi = mid，而实际上 p_true < threshold）
- **悲观偏差**：采样估计 `p̂` 可能低估 `p_true`，导致二分搜索拒绝一个实际可达标的资源值（lo = mid，而实际上 p_true >= threshold）

误差传播：单步的 ±ε_sampling 可能导致最终的 `min_resource` 偏离真实值。误差的大小取决于：
- N（模拟次数/迭代）—— N 越大，`p̂` 越精确
- 真实 p 离阈值越近，噪声导致的误判概率越高
- epsilon（精度）—— 噪声在没有精度约束时会放大误差传播距离

## 二、已知局限性

### 2.1 每步固定 N 的样本量分配

与最多目标卡面板相同的问题——每步迭代使用相同的模拟次数 N，不考虑不同资源量下的方差异质性（heteroskedasticity）。接近阈值边界的迭代可能需要更大的 N 来减少噪声导致的误判。

### 2.2 epsilon 的单池假设

`_extract_cost_per_draw()` 只返回第一个有效池的单抽消耗。多池且各池成本不同时，epsilon 精度可能不准确。见 CLAUDE.md 已知问题 #3。

### 2.3 与 P8 面板合并的设计变迁

当前与最多目标卡面板共享核心搜索模式（迭代模拟 → 成功率评估），但代码独立维护。P8 面板合并计划将二者整合为统一的多目标搜索面板。在合并后，二分搜索/翻倍法可抽象为通用搜索策略。

## 三、与外部理论的对照

| 概念 | 文献 | 一致性 |
|------|------|--------|
| 翻倍法上界搜索 | Bentley & Yao (1976), "An Almost Optimal Algorithm for Unbounded Searching" | ✅ 正确——在无噪声确定性函数上最优 |
| 二分搜索 | 经典 (Knuth, 1998, TAOCP §6.2.1) | ✅ 正确——在有噪声判定下需注意误分类 |
| 噪声条件下的最优搜索 | Burnashev & Zigangirov (1974), sequential testing theory | ❌ 未采用——当前仍为固定 N 的噪声判定 |
| 成功率单调性 | 随机优势（stochastic dominance） | ⚠️ 假设但未验证 |
