# 技术栈

> 最后更新：2026-06-02 | 基于 `_version.py` v1.10.0 + CLAUDE.md

## 运行时

- **语言**：Python 3.10+
- **GUI**：PyQt6 ≥ 6.0
- **数值计算**：numpy ≥ 1.20, scipy ≥ 1.10
- **绑图**：matplotlib ≥ 3.5（旧路径）、Plotly（新路径，通过 ChartSpec → PlotlyRenderer → ChartWebView）

## 开发工具

- **测试**：pytest ≥ 7.0, pytest-cov ≥ 4.0
- **构建**：setuptools ≥ 45, wheel

## 架构层

| 层 | 职责 | 依赖 |
|------|------|------|
| `core/` | 核心引擎——池子、状态、策略、保底、GDR、分析算法 | numpy |
| `service/` | 业务层——GachaService、batch_simulator、分析服务 | core |
| `gui/` | PyQt6 面板 + Worker 线程 | core + service |
| `generator/` | 排期/目标生成器 | core |
| `visualization/` | matplotlib 中文字体配置（旧） | matplotlib |
| `config/` | 默认配置文件（竖线分隔文本格式） | — |

## 并行架构

- `multiprocessing.Pool(initializer=_wk_init, initargs=(...))` 模式
- Worker 通过模块级全局变量接收 SimulationEnv（未来将合并为单个 `_wk_env`）
- 单进程路径（`max_workers <= 1`）直接在主进程调用
- GUI 面板使用 QThread + Worker 模式执行模拟
