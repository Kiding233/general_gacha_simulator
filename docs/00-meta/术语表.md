# 术语表

> 最后更新：2026-06-02

## 模拟领域

| 术语 | 缩写 | 定义 |
|------|------|------|
| 广义出率 | GDR | Generalized Drop Rate——将目标达成、资源效率、加权综合等统一度量的指标体系，共 13 种指标 |
| 保底 | Pity | 抽卡系统中保证玩家一定次数后获得稀有卡片的机制，分软保底（概率递增）和硬保底（100%触发） |
| 软保底 | Soft Pity | 计数器从 `start_at` 开始概率线性/指数/阶梯上升至 `end_at` 时 100% |
| 硬保底 | Hard Pity | 计数器达到 `threshold` 时 100% 触发 |
| 抽卡操作 | Draw | 从指定池子消耗资源抽取一张随机卡 |
| 等待操作 | Wait | 跳过一段时间，让时间线推进，影响池子开放/关闭 |
| 兑换 | Exchange | 使用资源直接换取目标卡（兑换池专用） |
| 紧凑结果 | CompactResult | dataclass，包含逐抽序列 + 汇总统计 + 池快照（主流输出格式） |
| 信息向量 | InfoVector | 逐抽详细记录（完整模式输出，已很少使用） |
| 流式分析 | Streaming | 边模拟边提取边丢弃，内存与模拟次数 N 无关 |
| 成功判定 | Success | GDR 值 ≥ 阈值（`lower_is_better` 指标时值 ≤ 阈值） |
| 目标卡 | Target Card | 用户指定的想要获得的卡 |

## 统计方法

| 术语 | 缩写 | 定义 |
|------|------|------|
| 自举法 | Bootstrap | 对已有 N 条结果做有放回抽样，估计统计量的置信区间 |
| 偏差校正加速法 | BCa | Bias-Corrected and accelerated Bootstrap——修正自举分布偏差和偏度 |
| 广义帕累托分布 | GPD | Generalized Pareto Distribution——用于尾部外推的极值分布 |
| 极值理论 | EVT | Extreme Value Theory——对分布尾部（最大值/最小值/分位数）建模 |
| 条件风险价值 | CVaR | Conditional Value at Risk——尾部期望值，超出 VaR 的条件期望 |
| 风险价值 | VaR | Value at Risk——给定置信水平下的分位数 |
| 核密度估计 | KDE | Kernel Density Estimation——非参数概率密度估计 |
| 局部逻辑回归 | LLR | Local Logistic Regression——局部加权二元响应回归 |
| 序贯概率比检验 | SPRT | Sequential Probability Ratio Test——达到决策边界时停止采样 |
| 假发现率 | FDR | False Discovery Rate——多重比较中控制错误发现比例 |
| 缩尾化 | Winsorization | 将极端值替换为分位数边界值（处理异常值的技术） |
| 马尔可夫链 | Markov Chain | 下一步状态仅依赖当前状态的随机过程 |

## 架构术语

| 术语 | 定义 |
|------|------|
| 面板 | Panel——GUI 中的一个 Tab 页，对应一个分析功能 |
| 子系统 | Subsystem——核心引擎层的一个功能模块 |
| 六文件 | 每个模块的标准文档集：00-档案 / 01-理论 / 02-实施 / 03-审计 / 04-问题 / 05-笔记 |
| 主题计划 | Theme Plan——一个完整的改进计划文件，存放在对应模块文件夹内，不被拆散 |
| 策略上下文 | StrategyContext——封装策略决策所需全部信息的 dataclass |
| 模拟环境 | SimulationEnv——一次批量模拟的完整运行时配置 |
| ConfigStore | 配置数据存储器——所有用户可编辑配置的集中存储 |
| ChartSpec | 图表规格中间表示——解耦图表描述与具体绑图库 |

## GDR 指标分类

| 类别 | 指标 | 值域 |
|------|------|------|
| 目标达成 | `target_achievement`, `target_collection`, `all_targets`, `ssr_collection` | [0,1] |
| 资源效率 | `resource_remaining`, `resource_efficiency`, `extra_target`, `non_pity_draws`, `pity_draws` | 实数 |
| 加权综合 | `weighted_satisfaction`, `total_card_value`, `per_pool_draw_rate`, `weapon_character_ratio` | 实数 |
