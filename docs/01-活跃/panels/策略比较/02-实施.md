## 二、关键函数清单（当前 v1）

### 2.1 `ComparisonWorker` (line 26-141)

QThread 子类，在后台线程顺序运行多个策略的模拟。

| 成员 | 签名/类型 | 说明 |
|------|----------|------|
| `progress` | `pyqtSignal(str, int)` | 进度信号（消息 + 百分比） |
| `finished` | `pyqtSignal(object)` | 完成信号（携带 results dict） |
| `error` | `pyqtSignal(Exception)` | 错误信号 |
| `__init__` | `(strategy_keys: list, num_simulations: int, config_store: ConfigStore, max_workers: int = 4)` | 初始化参数 |
| `stop` | `()` | 设置 `_should_stop` 标志 |
| `run` | `()` | **主逻辑**：构建 env → 遍历策略 → run_batch_parallel → 汇总 |
| `_compute_summary` | `(histories, display_name)` | 从模拟历史计算 5 个汇总指标 |

`run()` 的核心循环：

```python
for i, skey in enumerate(strategy_keys):
    entry = STRATEGY_REGISTRY.get(skey)
    display_name = entry['display_name'] if entry else skey

    # 当前配置策略使用配置中的参数，其他策略使用空参数
    if skey == current_strategy_key:
        sparams = current_strategy_params
    else:
        sparams = {}

    histories = run_batch_parallel(
        env=sim_env,
        target_specs=target_specs,
        initial_resources=sim_env.initial_resources,
        num_simulations=self.num_simulations,
        max_workers=self.max_workers,
        seed=0,                       # 所有策略共享 seed=0
        strategy_name=skey,
        strategy_params=sparams,
    )
    results[skey] = self._compute_summary(histories, display_name)
```

### 2.3 `_compute_summary()` 返回结构

```python
{
    'display_name': str,          # 策略中文显示名
    'count': int,                 # 有效模拟次数
    'avg_draws': float,           # 平均总抽卡数
    'avg_pity_triggers': float,   # 平均保底触发次数
    'avg_target_acquired': float, # 平均目标卡获得数
    'avg_final_resource': float,  # 平均最终 draw_resource
    'success_rate': float,        # 成功率 = 至少获得1张目标卡的比例
}
```

## 六、策略注册表（STRATEGY_REGISTRY）

当前注册 7 种策略，其中 `draw_target` 为内部策略不显示：

| key | 显示名 | 描述 |
|-----|--------|------|
| `smart` | 按需追卡 | 优先兑换 → 按目标追卡 → 等待 |
| `pool_quota` | 指定池配额 | 每池指定抽卡上限后切换 |
| `pity_reserve` | 保底预留 | 只在保底概率 >= 阈值时抽卡 |
| `stop_on_target` | 目标即停 | 抽到 up/目标卡后停止 |
| `target_hunting` | 指定池追卡 | 只在指定池子抽卡 |
| `fixed_count` | 固定次数 | 抽满固定次数后等待 |
| `draw_target` | （内部） | 无条件重复从目标池抽卡 |
