## 二、数据流（v2 当前）

```
批量模拟完成 → on_simulation_finished()
  ├─ 构建 ComparabilityFingerprint（9 维度）
  ├─ 构建 StoredDataset（原始数据 + fingerprint + GDRContext 序列化）
  └─ ResultStore.add() → datasets_changed 信号
       │
       ▼
数据管理 Tab（data_manager_panel.py）
  ├─ 表格展示所有数据集（名称/策略/目标卡/初始资源/N/时间/备注）
  ├─ 勾选 ≥2 → 可比性检查面板显示维度差异矩阵
  ├─ 点击「开始比较」→ compare_requested.emit(names)
  │     │
  │     ▼
  │   MainWindow._on_compare_datasets(names)
  │     ├─ 切换到「比较分析」Tab
  │     └─ comparison_analysis_panel.set_datasets(names)
  │          │
  │          ▼
  │        ComparisonWorker(QThread) ← 后台计算
  │          ├─ compute_gdr_values_for_datasets() → 提取每数据集逐次 GDR
  │          ├─ L1: DescriptiveStats.compute() × N → 均值/中位数/std/偏度/CVaR/success_rate
  │          ├─ L2: compute_dominance_matrix() × 3 (FSD/SSD/TSD, n_bootstrap=1000)
  │          └─ L3: compute_pvalue_matrix() → raw + Holm + BH
  │               │
  │               ▼
  │          finished.emit(results) → 主线程更新 UI
  │            ├─ _update_l1: 描述统计表 + ECDF/PMF Plotly 图表
  │            ├─ _update_l2: FSD/SSD/TSD HTML 矩阵（带表格线 + 颜色编码）
  │            ├─ _update_l3: p 值矩阵 (QTableWidget, 颜色编码 + Holm/BH/raw)
  │            └─ (L4 手动触发) _run_l4: 帕累托前沿 Plotly 散点图
```

## 三、关键函数清单（v2）

### 3.1 `ResultStore` (result_store.py)

| 成员 | 签名/类型 | 说明 |
|------|----------|------|
| `add` | `(name: str, dataset: StoredDataset) -> str` | 添加数据集（名称冲突追加后缀） |
| `remove` | `(name: str) -> bool` | 删除数据集 |
| `rename` | `(old: str, new: str) -> bool` | 重命名数据集 |
| `get` | `(name: str) -> Optional[StoredDataset]` | 按名称获取数据集 |
| `list_datasets` | `() -> List[Dict]` | 返回元数据列表（不含原始数据） |
| `set_current` | `(name: Optional[str])` | 设置当前加载的数据集 |
| `compare_fingerprints` | `(names: List[str]) -> Optional[ComparabilityDiff]` | 全对比较 2+ 数据集的可比性 |
| `save_all` | `(path: str)` | JSON 持久化全部数据集 |
| `load_all` | `(path: str)` | 从 JSON 加载全部数据集 |
| `get_cached` / `put_cached` | `(name, cache_key) / (name, cache_key, result)` | 分析缓存存取 |

信号：`datasets_changed`、`current_changed(str)`

### 3.2 `ComparabilityFingerprint` (result_store.py)

9 维度指纹：`config_hash`、`strategy_name`、`target_cards`、`initial_resources`、`stop_condition`、`seed_start`、`seed_end`、`num_simulations`、`pool_ids`、`created_at`

### 3.3 `ComparabilityDiff` (result_store.py)

| 方法 | 说明 |
|------|------|
| `only_strategy_differs()` → bool | 仅策略不同的纯策略比较 |
| `all_same()` → bool | 全部相同——变异分析 |
| `mode_label()` → str | 返回 5 种比较模式的中文标签 |

### 3.4 `ComparisonAnalyzer` 成员 (comparison_analyzer.py)

| 函数/类 | 签名 | 说明 |
|---------|------|------|
| `DescriptiveStats.compute` | `(name, gdr_key, values, threshold, lower_is_better) -> DescriptiveStats` | L1 描述统计 |
| `compute_gdr_values_for_datasets` | `(datasets, gdr_key, target_specs_list, threshold) -> (values_list, names, lower_is_better)` | 提取多数据集逐次 GDR |
| `dd_bootstrap_test` | `(samples_a, samples_b, order, n_bootstrap, rng_seed) -> Dict` | DD Bootstrap 随机占优检验 |
| `compute_dominance_matrix` | `(values_list, names, order, n_bootstrap, rng_seed) -> Dict` | n×n 占优矩阵 |
| `HypothesisTestResult.from_samples` | `(samples_a, samples_b, method, lower_is_better, alpha) -> HypothesisTestResult` | L3 单对假设检验 |
| `compute_pvalue_matrix` | `(values_list, names, method, lower_is_better, alpha) -> Dict` | n×n p 值矩阵 + 三种校正 |
| `holm_bonferroni` | `(p_values: List[float]) -> List[float]` | Holm-Bonferroni FWER 校正 |
| `benjamini_hochberg` | `(p_values: List[float]) -> List[float]` | Benjamini-Hochberg FDR 校正 |
| `ParetoFrontier.compute` | `(x_values, y_values, names, x_gdr, y_gdr, x_lib, y_lib) -> ParetoFrontier` | L4 双 GDR 帕累托前沿 |

### 3.5 `ComparisonWorker` (comparison_analysis_panel.py)

QThread 子类，在后台线程计算 L1+L2+L3。

| 信号 | 说明 |
|------|------|
| `progress(str)` | 进度消息 |
| `finished(dict)` | 完成，携带 `values_list / names / lower_is_better / stats_list / dom_results / pmat / correction` |
| `error(str)` | 错误消息 |

### 3.6 `ComparisonAnalysisPanel` (comparison_analysis_panel.py)

| 方法 | 说明 |
|------|------|
| `set_datasets(names)` | 设置数据集列表并自动运行分析 |
| `_run_analysis()` | 创建 ComparisonWorker → 后台计算 |
| `_update_l1(stats_list, values_list, names, lower_is_better)` | L1 描述统计表 + ECDF/PMF Plotly |
| `_update_l2(dom_results, names)` | L2 FSD/SSD/TSD HTML 矩阵 |
| `_update_l3(pmat, names, correction)` | L3 p 值矩阵 |
| `_run_l4()` | L4 帕累托前沿 Plotly 散点图（手动触发） |

## 七、策略注册表（STRATEGY_REGISTRY）

当前注册 7 种策略，其中 `draw_target` 为内部策略不显示：

| key | 显示名 | 描述 |
|-----|--------|------|
| `smart` | 按需追卡 | 优先兑换 → 按目标追卡 → 等待 |
| `pool_quota` | 指定池配额 | 每池指定抽卡上限后切换 |
| `pity_reserve` | 保底预留 | 只在保底概率 >= 阈值时抽卡 |
| `stop_on_target` | 目标即停 | 抽到 up/目标卡后停止 |
| `target_hunting` | 指定池追卡 | 只在指定池子抽卡 |
| `fixed_count` | 固定次数 | 抽满固定次数后等待 |
| `draw_target` | （内部） | 无条件重复从目标池抽卡 |
