# 策略比较 — 理论

> 建档日期：2026-06-02 | 更新：2026-06-04 | 版本：v2

---

## 一、核心假设

### A1：各策略运行于相同的评估环境

当前实现中，每个策略使用相同的 `ConfigStore` → `SimulationEnv` 运行。这确保比较的有效性——所有策略面对相同的：

- 池子序列（开放时间、持续时间、卡牌分布）
- 保底规则（软保底/硬保底参数、重置条件）
- 资源结构（初始资源、每日收入、兑换汇率）
- 目标函数（目标卡及其数量）

若环境不同，策略比较无意义。P19 的重构将环境一致性的保证从「同一 ConfigStore」扩展到「任意两个数据集的配置可比性检查」。

### A2：蒙特卡洛模拟可充分代表策略长期表现

以 `num_simulations`（默认 500）次独立模拟的样本均值估计真实期望值。这依赖大数定律——N 足够大时样本均值收敛于真实期望。

**脆弱环节**：

- N=500 时，均值的标准误差约为 σ/√500 ≈ 0.045σ——对于高方差策略（如全押策略），均值估计可能仍有数个百分点的不确定性
- 极端百分位数（如 CVaR₉₅）在 N=500 下的估计精度更差（只有约 25 个样本落在 95% 分位数之外）
- 当前实现不报告任何置信区间——点估计缺乏不确定度量

### A3：随机种子独立（当前实现）

当前每个策略使用 seed=0 运行——即所有策略共享相同的随机种子初始化，但模拟路径因策略决策不同而分叉。这不是真正的配对设计：

- 同一随机种子下，若策略 A 在池 1 抽了 50 次而策略 B 只抽了 10 次，它们消耗了随机数序列的不同部分
- 时间线差异导致「同一随机种子」不等价于「相同的运气」

P19 计划引入的配对模式需要更严格的控制：确保在可比的时间点使用相同的随机数初始化。

### A4：成功率现由 GDR 框架统一判定 ✅

P19 Phase 2 已解决此问题。`ComparisonAnalyzer` 通过 `compute_gdr_values_for_datasets()` 使用 GDR 框架统一判定成功/失败。用户可自由选择 13 种 GDR 指标和自定义阈值。旧 `_compute_summary()` 已随 `strategy_comparison_panel.py` 删除。

### A5：指标独立性

当前比较的 5 个指标（平均抽卡数/保底触发/目标获得/剩余资源/成功率）被视为独立维度。但实际上它们之间存在强相关性：

- 高抽卡数 → 高目标获得 → 低剩余资源
- 低保底触发 → 可能意味着「运气好早早出货」→ 低抽卡数 + 高剩余资源

P19 的帕累托分析将显式处理这些协同/拮抗关系。

## 二、已知局限性

### ~~L1：无统计显著性检验~~ ✅ 已解决 (P19 Phase 2)

L3 假设检验层支持 KS/MWU/ttest + p 值矩阵。L2 DD Bootstrap 随机占优 (FSD/SSD/TSD)。用户可查看颜色编码的显著性。

### ~~L2：无多重比较校正~~ ✅ 已解决 (P19 Phase 2)

Holm-Bonferroni (FWER) 和 Benjamini-Hochberg (FDR) 已实现。用户可在面板切换校正方式。

### L3：无配对/反事实分析 ⏸ 暂缓 (Phase 3)

当前若两个数据集种子区间和 N 相同，可比性检查会标注「变异分析」。正式配对模式（配对 t 检验 + 反事实遗憾）暂缓至 Phase 3。

### ~~L4：无配置差异比较~~ ✅ 已解决 (P19 Phase 1)

ResultStore 使比较对象从「策略列表」扩展为「任意数据集组合」。可比性检查自动识别 4 种比较模式。

### ~~L5：无持久化/导入导出~~ ✅ 已解决 (P19 Phase 1)

ResultStore.save_all() / load_all() JSON 持久化 + 数据管理面板导入/导出按钮。

### L6：无外部库假设检验

当前无任何统计检验实现。P19 附录已确认 scipy（已依赖）提供 KS/MWU/置换检验/配对 t 检验，statsmodels 提供多重比较校正，pingouin 提供 Cohen's d——无需从零手写。

## 三、统计框架（P19 目标状态）

### 3.1 假设检验方法

| 方法 | 零假设 | 适用模式 | 外部实现 |
|------|--------|---------|---------|
| 置换检验 | 两个策略在统计量 λ 上无差异 | 分布模式 | `scipy.stats.permutation_test` |
| KS 检验 | 两个策略的指标分布相同 | 分布模式 | `scipy.stats.ks_2samp` |
| Mann-Whitney U | 两个策略的指标分布无随机占优 | 分布模式 | `scipy.stats.mannwhitneyu` |
| 配对 t 检验 | 配对差值的均值为 0 | 配对模式 | `scipy.stats.ttest_rel` |
| Wilcoxon 符号秩 | 配对差值的中位数为 0 | 配对模式 | `scipy.stats.wilcoxon` |
| Cohen's d | （效应量，非假设检验） | 两种模式 | `pingouin.compute_effsize` |

### 3.2 随机占优

设 F_A 和 F_B 为策略 A 和 B 在某指标上的 CDF。

- **一阶随机占优 (FSD)**：F_A(x) <= F_B(x) 对所有 x，且至少一处严格不等。A FSD B 意味着任何非递减效用函数的决策者都偏好 A 于 B。
- **二阶随机占优 (SSD)**：∫F_A(t)dt <= ∫F_B(t)dt 对所有 x。A SSD B 意味着任何风险厌恶的决策者都偏好 A 于 B。

### 3.3 多重比较校正

对 m = C(k,2) 对比较的 p 值进行调整：

| 方法 | 控制目标 | 保守程度 |
|------|---------|---------|
| 无校正 | — | 过高估计显著性 |
| Holm-Bonferroni | 族系错误率 (FWER) | 保守 |
| Benjamini-Hochberg | 假发现率 (FDR) | 中等（推荐默认） |
