## 概述

**暂无活跃问题。**

批量模拟面板是项目中连接配置和分析的「管道」组件，逻辑相对简单。当前无已知 P0/P1/P2 级别问题。

## 搁置条目引用

以下条目来自 `docs/00-meta/搁置计划记录.md`，与本模块相关但非活跃问题：

- **Worker 全局变量消除方案**（`docs/03-归档/plans/Worker全局变量消除方案.md`）：将 11 个 `_wk_*` 模块级全局变量合并为 1 个 `_wk_env: SimulationEnv`。此重构主要影响 `service/batch_simulator.py`，批量模拟面板仅作为调用方受益于接口简化。
- **多面板 CPU 争抢**（CLAUDE.md 已知问题 #2）：各面板 Worker 线程无全局协调，可能出现多个面板同时触发模拟导致 CPU 饱和。批量模拟面板是 CPU 密集操作的主要触发者。
