## 一、脆弱性分析的理论假设

### 1.1 核心假设

| 假设 | 内容 | 成立条件 |
|------|------|---------|
| H1: 局部平滑 | P(失败|资源=r) 作为 r 的函数在局部近似线性（或经 logit 变换后近似线性） | 带宽适当；数据密度足够；无大规模间断 |
| H2: 二元响应 | 每次模拟产出二值结局（成功/失败），由 GDR 阈值判定 | 阈值选择合理；GDR 指标恰当 |
| H3: 条件独立性 | 给定资源 r，各次模拟的失败判定独立 | 模拟间独立（不同 seed 保证）；但同一模拟内多池不独立 |
| H4: 单调性（未强制） | P(失败|r) 随 r 递减——资源越多失败概率越低 | 经济直觉支持，但算法不强制单调 |
| H5: 设计点视为固定 | 将资源观测值视为固定设计点（design points），忽略其随机性 | 资源值本身是随机的（抽样变异），当前将其视为固定协变量，可能引入偏差 |
| H6: 正态核 | 使用高斯核进行局部加权 | 标准选择；对于有界资源 (≥0)，边界处核支撑超出定义域需处理 |

### 1.2 当前方法

- **主方法**：局部线性回归（LLR, local linear regression），p=1 阶局部多项式 + 高斯核加权
- **备选方法**（阶段一已实现）：局部线性逻辑回归（local linear logistic），通过 IRLS 最大化局部加权对数似然
- **带宽选择**：Silverman 规则 `h = 1.06·σ·n^(-1/5)`——此为核密度估计最优带宽，非回归最优带宽
- **方差估计**：高斯核平滑残差方差（非参数）
- **脆弱区间检测**：`lp_curve > alpha` 逐点阈值 + 连续区间提取

## 四、假设违反的后果速查

| 假设 | 违反情景 | 后果 |
|------|---------|------|
| H1: 局部线性 | P(失败|r) 在过渡带急剧转折（如 0→1 在 200 抽内） | 转折被过度平滑，脆弱区间边界模糊 |
| H2: GDR 阈值判定 | GDR 阈值取中间值（如 0.7），产生大量「部分成功」样本 | 二元化丢失信息；应考虑序数 logistic 回归或直接建模 GDR 连续值 |
| H3: 条件独立性 | 同一模拟内多池间存在策略依赖 | 方差估计偏小（忽略组内相关性），置信带偏窄 |
| H6: 高斯核边界 | 资源接近 0（定义域左边界） | 核支撑超出定义域，左边界估计向数据内部收缩（问题 B） |
