## P0：转变分析阶段一 + 每池GDR阶段一（高优先级，立即可见的严谨性提升）

### 覆盖缺陷

**转变分析（P27 阶段一）**：
- **T1**（严重）：转移概率无置信区间——Wilson 得分区间 + UI CI 展示
- **T2**（中等）：零分母静默归零——NaN + low_sample 标记
- 连续度量附加（`mean_resource_consumed_given_success`/`_failure`）

**每池GDR（P28 阶段一）**：
- **P2**（严重）：Simpson 悖论诊断——CV + 极差自动检测 + UI 异质性指示器
- **P4**（中等）：BA 似然比 inf→NaN 修复 + UI 处理

### 涉及文件

| 文件 | 变更内容 |
|------|---------|
| `core/per_pool_analysis.py` | TransitionMatrix 新增 ci_lower/ci_upper、low_sample 字段、连续度量字段 |
| `core/process_analysis.py` | BA ratio 分母为 0 → NaN + low_sample |
| `gui/analysis_panel.py` | 转变热图 CI 展示 + NaN 显示「数据不足」+ Simpson 异质性指示器 |

### 预期工作量

2-3 天（转变分析）+ 1-2 天（每池GDR）= 3-5 天合计

### 风险

- Wilson CI 在极端概率（p=0 或 p=1）时可能产生单侧区间，UI 需处理
- Simpson CV 阈值 0.3 为经验值，需在实际数据上验证灵敏度和特异度

## P1：转变分析连续度量与马尔可夫假设诊断（中优先级）

### 覆盖缺陷

**转变分析（P27 阶段二）**：
- **T3**（严重）：马尔可夫假设诊断——资源消耗差异 >30% 时 UI 警告 + 保底跨池继承时标注
- **T4**（中等）：多池转移 FDR 校正 + Fisher 精确检验
- **T5**（低等）：事件-成败与池-池转变 UI 双向交叉引用说明

### 涉及文件

| 文件 | 变更内容 |
|------|---------|
| `gui/analysis_panel.py` | 马尔可夫假设警告标签 + FDR 校正可选开关 + 交叉引用说明文字 |
| `gui/process_analysis_panel.py` | 对称交叉引用说明（从过程分析指向转变分析） |
| `core/per_pool_analysis.py` | FDR Benjamini-Hochberg 程序（可选） |

### 预期工作量

2-3 天

### 风险

- FDR 校正假设检验间正相关（Benjamini & Yekutieli, 2001），若池间负相关（资源竞争），FDR 控制可能保守
- Fisher 精确检验在池对数量多时计算开销递增（K 个池 → K 个 2×2 表）

## P2：Bootstrap CI 依赖与深化改进（中优先级，有外部依赖）

### 覆盖缺陷

**转变分析（P27 阶段三）**：
- T1 深化：Bootstrap 同时置信域（Sison-Glaz 方法）替代 Wilson CI

**每池GDR（P28 阶段二）**：
- **P1**（严重）：Bootstrap 同时置信域——池间相关结构保留的联合 CI
- **P3**（中等）：经验贝叶斯收缩（Beta MLE 先验 + James-Stein 收缩）
- **P5**（中等）：pseudo_compact 补全 + GDR 依赖声明

### 外部依赖

- **P3（Bootstrap 引擎）**：多项比例同时置信域直接依赖 P3 的重抽样基础设施——P3 完成后本项方可启动
- **P18（Bootstrap 修复）**：P18 的 BCa CI 方法可复用于二项比例 CI

### 涉及文件

| 文件 | 变更内容 |
|------|---------|
| `core/per_pool_analysis.py` | Bootstrap 同时置信域 + FDR Benjamini-Hochberg 程序 |
| `core/process_analysis.py` | 经验贝叶斯收缩（Beta MLE 先验拟合 + 后验均值） |
| `core/process_trace.py` | pseudo_compact 字段补全 + GDR 依赖声明元数据 |
| `gui/analysis_panel.py` | 同时置信域误差条/置信椭圆 + 经验贝叶斯标注 |

### 预期工作量

3-5 天

### 风险

- Sison-Glaz 同时置信域在概率接近边界值（0 或 1）时数值求解可能不稳定
- 经验贝叶斯收缩假设事件类型间存在可共享的先验——若事件类型间异质性极高，层次先验的拟合质量可能不佳

| 优先级 | 判断标准 |

| **P0** | 无外部依赖；缺陷严重度为「严重」；修复后立即可见的用户体验和严谨性提升 |
