## P0：P26 阶段一 —— 低风险快速修复

**对应计划**：`P26 退路搜索与资源目标搜索理论改进计划.md` 第三章阶段一

**整体说明**：四项 UI/算法层面的低风险修复，不改变核心算法结构，收益明确、风险可控。建议优先于 P15 阶段二之前完成，因为这些修复涉及用户直接感知的行为（Pareto 声称降级 + epsilon 调整 + 权重标注 + 上界统一）。

| 子项 | 简称 | 涉及文件 | 修复内容 |
|------|------|---------|---------|
| P0-1 | Pareto 声称降级 | `retreat_search_panel.py`, `strategy_panel.py` | UI 文案「Pareto 前沿」→「资源-目标权衡曲线（贪心近似）」+ 支配性过滤（标记被支配点，UI 可切换）+ Tooltip 说明 |
| P0-2 | 翻倍上界统一 | `retreat_search.py`, `resource_search_panel.py` | 统一为 15 次翻倍 + 提取 `_find_upper_bound()` 公共函数 + 二分结束后检测上界不足 |

**对应计划**：
- P15 阶段二（统计严谨性提升）：RSC 带宽 + Wild Bootstrap 同时置信带 + persistence 过滤

- P26 阶段二（SPRT 二分搜索）：SPRT 决策模块 + 替换固定 N 采样

### P1-A：P15 阶段二 —— RSC 带宽选择

| 属性 | 内容 |
|------|------|
| 涉及文件 | `core/vulnerability.py` |
| 预计工作量 | 1-2 天 |
| 依赖 | 无（独立于其他修改） |
| 风险 | 中——RSC 的 O(n²) 留一估计在 n>5000 时需降级为 GCV 近似 |
| 文献 | Fan & Gijbels (1995) 残差平方准则 |

### P1-B：P15 阶段二 —— Wild Bootstrap 同时置信带

| 属性 | 内容 |
|------|------|
| 涉及文件 | `core/vulnerability.py` |
| 预计工作量 | 2-3 天 |
| 依赖 | 局部 logistic 回归（阶段一已完成）效果最好 |
| 风险 | 中——B=999 × IRLS 迭代的耗时，需提供 `fast=True` 渐近近似模式 |
| 文献 | Härdle & Mammen (1991); Wasserman (2006) Ch.5 |
| 备注 | 与 P3（Bootstrap 稳定性分析）互补但不重叠——P3 是对估计量稳定性做 Bootstrap，此处是对同时置信带做 Bootstrap |
