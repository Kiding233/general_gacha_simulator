## 设计 ↔ 实施对齐检查

| 设计项 | 实施状态 | 备注 |
|--------|---------|------|
| 3 种 QRadioButton 搜索模式 | ✅ | `mode_group` QButtonGroup |
| 起始池 QComboBox（含 "(从头开始)"） | ✅ | `pool_combo` |
| 灰显替代消失（setEnabled） | ✅ | `_on_mode_changed()` / `_on_pool_changed()` |
| 统一双列权重表 | ✅ | `weight_table`（抽取意愿 + 错失代价） |
| 搜索方向 QRadioButton | ✅ | `dir_forward` / `dir_backward` |
| Pareto 默认后退可切前进 | ✅ | `_on_mode_changed()` 设 `dir_backward.setChecked(True)` |
| 结果快照系统 | ✅ | `SnapshotManager` |
| 统一错误处理 | ✅ | `DetailedError` 包装 + `traceback.print_exc()` |
| 引擎统一停止 | ✅ | `engine.stop()` |
| cost_per_draw 归并 | ✅ | 模块级 `get_cost_per_draw()` |
