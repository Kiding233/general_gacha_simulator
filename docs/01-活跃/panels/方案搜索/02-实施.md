### PlanSearchResult 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `search_mode` | `str` | `'min_resource'` / `'forward'` / `'backward'` / `'pareto'` |
| `direction` | `str` | 仅 Pareto 模式：`'forward'` / `'backward'`（默认 `'backward'`） |
| `from_pool_id` | `Optional[str]` | `None`=完整时间线，`str`=截断池ID |
| `points` | `List[RetreatSearchPoint]` | 搜索路径上的所有评估点 |
| `start_mode` | `str` | `'full_timeline'` / `'from_retreat'`（`__post_init__` 自动推断） |
| `min_resource` | `Optional[float]` | 最少资源结果（仅 `min_resource` 模式） |
| `binary_steps` | `List` | 二分搜索步骤记录 |
| `pareto_points` | `List[RetreatSearchPoint]` | Pareto 前沿点集 |
| `success` | `bool` | 搜索是否成功（找到满足阈值的方案） |
| `cost_per_draw` | `float` | 单抽成本（默认 160） |
| `total_iterations` | `int` | 总迭代次数 |
| `final_success_probability` | `float` | 最终方案的成功率 |

### 3.4 主面板

```python
class PlanSearchPanel(QWidget):
    status_update = pyqtSignal(str)

    # 5 个 QGroupBox
    _setup_mode_group()     # ① 搜索模式：3 个 QRadioButton（最少资源/最多目标卡/Pareto）
    _setup_source_group()   # ② 起始状态：池下拉 + 基准资源 + 保底水位表
    _setup_weight_group()   # ③ 目标卡权重：双列表（抽取意愿/错失代价）+ 搜索方向
    _setup_params_group()   # ④ 搜索参数：GDR/阈值/模拟次数/并行/二分参数
    _setup_action_bar()     # ⑤ 运行/停止按钮 + 进度条 + 状态标签

    # 灰显逻辑
    _on_mode_changed()      # 最少资源→权重灰显；最多目标卡→二分灰显；Pareto→方向默认后退
    _on_pool_changed()      # 从头开始→基准资源/保底灰显

    # 外部接口
    set_store(ConfigStore)                     # 填充权重表 + 策略标签 + 起始池下拉
    set_vulnerability_result(VulnResult)       # 接收脆弱性分析结果，填充退路点选项
```
