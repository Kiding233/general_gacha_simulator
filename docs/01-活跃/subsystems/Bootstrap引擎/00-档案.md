## 业务问题

所有统计估计（GDR 均值/中位数/分位数、各池成功率、资源消耗均值、过程分析各类概率等）只有一个点估计，缺乏不确定性量化。决策者无法判断「这个 72% 的成功率是基于 100 次模拟还是 10,000 次模拟」，也看不到点估计周围的置信区间。

传统做法是增加模拟次数 N 来降低标准误，但这增加了计算成本。Bootstrap 在**已有的 N 条模拟结果上做有放回重抽样**，无需重新运行模拟，零额外模拟成本即可获得置信区间。

## 设计意图

- **输入**：模拟完成后已经存储的 N 条个体结果（成功/失败标志、GDR 值数组、资源消耗数组等）
- **处理**：`BootstrapEngine` 对输入数组做 B 次有放回重抽样，每次计算目标统计量，B 组 Bootstrap 复制值的经验分位数即置信区间
- **输出**：`BootstrapResult` dataclass，包含点估计、CI 上下界、Bootstrap 标准误、方法名称、样本量、重抽样次数
- **默认方法**：BCa（偏差校正加速）方法——校正百分位法的偏差和偏态，二阶精度 O(1/n)。BCa 失败时自动回退百分位法
- **特殊场景**：厚尾分布使用 m-out-of-n Bootstrap；极端尾部分位数（q <= 0.1）使用 GPD 参数 Bootstrap

## 技术实现

### 调用链

```
面板 Worker._run_impl()
  → BootstrapEngine(pools, state, schedule).run_simulation(initial_state, collector=CompactCollector())
  → result_collector.on_result(compact)
  → BootstrapEngine.bootstrap_probability(success_flags) / bootstrap_mean(values) / bootstrap_quantile(values, q)
  → scipy.stats.bootstrap (BCa/percentile) 或 scipy.stats.genpareto (GPD 尾部)
  → BootstrapResult{point_estimate, ci_lower, ci_upper, bootstrap_std, method, n_samples, b_replicates}
```

### 核心类

| 类 | 职责 |
|----|------|
| `BootstrapResult` | 不可变 CI 结果 dataclass，7 个字段 + `format_ci()` 格式化方法 |
| `BootstrapEngine` | 重抽样引擎，B=1000, ci_level=0.95, random_seed=42 |

### 核心方法

| 方法 | 用途 | 委托 |
|------|------|------|
| `bootstrap_probability(success_flags, use_bca=True)` | 二分类成功概率 CI | `scipy.stats.bootstrap(method='BCa')` |
| `bootstrap_mean(values, use_bca=True, auto_heavy_tail=False)` | 连续量均值 CI | `scipy.stats.bootstrap(method='BCa')` 或 m-out-of-n |
| `bootstrap_quantile(values, q=0.5, use_gpd=False, auto_heavy_tail=False)` | 分位数 CI | `scipy.stats.bootstrap(method='percentile')` 或 `_bootstrap_tail_gpd()` |
| `detect_heavy_tail(data)` | 厚尾检测（Hill 估计量） | 内部 `hill_estimator()` |
| `_select_m(n)` | m-out-of-n 的 m 选择 | Bickel & Sakov (2008) 经验公式 m=n^(2/3) |
| `_bootstrap_tail_gpd(data, q)` | 尾部低分位数参数 Bootstrap | `scipy.stats.genpareto.fit()` + `genpareto.rvs()` |
| `hill_estimator(values, k=None)` | Hill 尾部指数估计 | 自动拐点检测（`_select_hill_k()`） |
| `total_variation_distance(p, q)` | 两离散分布 TVD | 纯 numpy |

### 外部依赖

- `scipy.stats.bootstrap`：提供 BCa 和 percentile 方法
- `scipy.stats.genpareto`：GPD 拟合和抽样，用于参数 Bootstrap

### 与分布估计（EVT）的关系

`_bootstrap_tail_gpd()` 当前直接调用 `genpareto.fit()` 自行拟合 GPD，与 `evt_tail.py` 中的 GPD 拟合逻辑重复。计划（B2.8）改为委托 `evt_tail.fit_gpd_lower()` 统一拟合路径。
