### BootstrapResult (`bootstrap.py` line 17-30)

```python
@dataclass
class BootstrapResult:
    point_estimate: float
    ci_lower: float
    ci_upper: float
    bootstrap_std: float
    method: str = 'percentile'
    n_samples: int = 0
    b_replicates: int = 0

    def format_ci(self, precision: int = 3) -> str:
        """格式化为「点估计 [下界, 上界]」"""
```

7 个字段，其中 `method` 取值为 `'BCa'`、`'percentile'`、`'GPD-param'`、`'m-out-of-n (m=..., auto: heavy tail ...)'` 之一。

## 异常处理策略

| 场景 | 处理 |
|------|------|
| 输入为空数组 (n=0) | 返回 NaN CI 的 `BootstrapResult` |
| BCa 计算失败（scipy 异常） | 静默回退 `percentile` 方法 |
| GPD 超额 < 20 | 回退标准 Bootstrap |
| `genpareto.fit()` 异常 | 回退标准 Bootstrap |
| Hill 估计输入全非正 | 返回 `inf`（非厚尾） |
| Hill 估计有效对数比率 < 3 | 返回 `inf` |
| `_select_hill_k` 范围内 k 不足 5 | 返回 `sqrt(n)` 作为默认 k |
