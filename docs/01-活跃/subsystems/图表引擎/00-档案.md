# 图表引擎 -- 档案

> 创建日期：2026-06-02（首次建档）
> 无关联主题计划（P11 迁移已完成）
> 关联模块：`visualization/chart_spec.py`、`visualization/plotly_charts.py`、`gui/chart_webview.py`
> 关联子系统：Bootstrap 引擎（CI 直方图）、分布估计（CDF 图）、所有 GUI 面板（各面板通过 ChartSpec → PlotlyRenderer → ChartWebView 管道渲染图表）

---

## 业务问题

抽卡模拟系统需要为 10 个分析面板生成多种类型的统计图表——直方图、CDF、山脊线图、箱线图、散点图、条形图、热力图、3D 瀑布图、子图网格、表格。传统方案（matplotlib 静态 PNG）存在以下问题：

1. **无交互性**——悬停提示、缩放、平移、局部数据查询均不可用
2. **文件 I/O 开销**——每次更新需写临时 PNG 文件、加载到 QLabel，切换图表需重载
3. **多图表管理复杂**——每个图表一个 QLabel，多个图表用 QTabWidget，切换延迟明显
4. **跨面板代码重复**——各面板独立实现绑图逻辑，样式不统一

需要一个**统一的图表引擎**，支持交互式渲染、多图表无延迟切换、以及面板-渲染解耦。

## 设计意图

### 三层架构

```
面板层 (gui/*.py)
  → 构建 ChartSpec (纯数据，不依赖任何绑图库)
    → PlotlyRenderer (core/plotly_charts.py)
      → go.Figure (plotly.graph_objects)
        → ChartWebView (QWebEngineView)
          → 浏览器渲染 (Plotly.js)
```

1. **ChartSpec 中间表示层**（`visualization/chart_spec.py`）——纯数据 dataclass，零外部依赖（仅 `numpy`）。面板构建 `ChartSpec` 对象，不直接接触 Plotly。支持 10 种图表类型。
2. **PlotlyRenderer**（`visualization/plotly_charts.py`）——`ChartSpec → go.Figure → JSON` 的唯一转换点。全项目唯一 `import plotly` 的模块。
3. **ChartWebView**（`gui/chart_webview.py`）——封装 `QWebEngineView`，管理 Plotly.js 生命周期。多图表通过 JS `display: block/none` 切换，瞬时无延迟。支持增量更新单个图表。

### 防闪烁策略

- `__init__` 中预加载骨架页（含 `plotly.js` + `rebuildAll` 函数，空图表）
- `set_charts()` 首次及后续均通过 `runJavaScript` 原地重建，**零重载**（无 `loadFinished` 等待）
- 若预加载未完成时调用 `set_charts()`，写入 `_pending_charts` 等待 `loadFinished` 处理

### 10 种图表类型

| ChartType | Data dataclass | 用途 |
|-----------|---------------|------|
| `histogram` | `HistogramData` | 分布直方图（GDR 值、资源消耗、抽数分布） |
| `cdf` | `CDFData` | 经验累积分布函数 |
| `ridge` | `RidgeData` | 山脊线图——多组样本按标签叠加（策略比较） |
| `boxplot` | `BoxplotData` | 箱线图——多组分布对比 |
| `scatter` | `ScatterData` | 散点/折线图（含多轨迹支持） |
| `bar` | `BarData` | 条形图（垂直/水平） |
| `heatmap` | `HeatmapData` | 热力图（二维分布） |
| `waterfall_3d` | `Waterfall3DData` | 3D 瀑布图（资源-池-计数） |
| `subplot_grid` | `SubplotGridData` | 子图网格（多 heatmap 合并） |
| `table` | `TableData` | Plotly 表格——替代 QTableWidget |

### 注解系统

- `ChartAnnotation`：参考线/标注（支持 `vline`/`hline`，可配置颜色、虚线样式、文本）
- `ShadedRegion`：半透明着色区间（用于标注脆弱区间、置信区间）

## 技术实现

### 图表渲染管道

```
面板准备数据
  → 构建 ChartSpec(chart_type="histogram", data=HistogramData(...), title="...")
  → renderer.build_figure(spec) → go.Figure
  → renderer.figure_to_json(fig) → JSON string
  → webview.set_charts([("tab_name", spec), ...])
    → 生成 HTML + JS 代码
    → webview.page().runJavaScript(js_code)
    → 浏览器渲染 Plotly 图表
```

### 便捷构造函数

`chart_spec.py` 提供便捷构造函数（`histogram()`、`cdf()`、`ridge()` 等），减少样板代码：

```python
from gacha_simulator.visualization.chart_spec import histogram, ChartAnnotation

spec = histogram(
    samples=gdr_values,
    title="GDR 分布直方图",
    xlabel="GDR 值",
    mean_line=True,
    quantile_lines=[0.05, 0.95],
    annotations=[ChartAnnotation(type="vline", value=threshold, text="阈值")],
)
```
